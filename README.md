# J3 Project
## Frontend Deployment
1. Open terminal
2. Change directory into the frontend folder
3. To run the website on localhost, run "npm run dev" on terminal.
```bash
npm run dev
```
Alternatively, use pnpm dev if pnpm is installed
```bash
pnpm dev
``` 
4. Open http://localhost:3000/ to view webpage
